This page dedicates towards the idea of "Experimental Learning", which by the name, suggests that
you could utilize experimentation while learning. What does that mean, you say? Simple, if you do remember the scientific method(or maybe you don't),
here is the basic steps:

1. Observation / Question

2. First, research the said topic / area for more information

3. Create a hypothesis / educated guess

4. Test with an experiment

5. Analyze the result

6. Reflect / Report conclusions
